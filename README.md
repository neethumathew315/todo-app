# 📝 ToDo App

A simple and clean ToDo List application built using **HTML**, **CSS**, and **JavaScript**.
Users can add tasks, mark them as completed, and delete them.

---

## 🚀 Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Press **Enter key** to add task
* Clean and responsive UI
* Focus returns to input after adding task

---

## 📂 Project Structure

```
todo-app/
│
├── index.html
├── style.css
└── script.js
```

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Vanilla JavaScript (DOM Manipulation)

---

## 💡 How It Works

1. User types a task in the input field
2. Click **Add List** button or press **Enter**
3. Task appears in the list
4. Click ✔ to mark complete
5. Click ✖ to delete task
